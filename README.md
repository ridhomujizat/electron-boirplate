# Boirplate for Create React App - Electron

boirplate for electron-react build with webpack & electron-forge

### How to use
 1. Clone this repo 
    ```
    git clone https://github.com/ridhomujizat/cra-electron.git
    ```
 2. Go to directory
    ```
    cd cra-electron
    ```
 4. Type and run
    ```
    npm install
    ```
 6. And then
    ```
    npm run dev
    ```
